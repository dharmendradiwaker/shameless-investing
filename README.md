# Indian Investors Investment Tracker

## Overview

This repository contains a tool to track the recent investments and current holdings of top investors in India. Additionally, it provides a function called get_company_data that allows users to retrieve a list of all stocks along with their full financial data as per their requirement.

## Features

- **Investor Investment Tracking**: Get updates on recent investments made by India's top investors.
- **Current Holdings**: Stay informed about the current holdings of these investors.
- **Financial Data Retrieval**: Function get_company_data fetches comprehensive financial data for all stocks based on user needs.

## Installation

To get started with the Indian Investors Investment Tracker, follow these steps:

1. Clone the repository to your local machine:

    ```bash
    git clone <repository_URL>
    ```

2. Install the required dependencies:

    ```bash
    pip install -r requirements.txt
    ```

3. Run the application:

    ```bash
    python app.py
    ```

## Usage

1. **Investment Tracking**:

    - Access the latest updates on recent investments and current holdings of top Indian investors.

2. **Financial Data Retrieval**:

    - Use the get_company_data function to retrieve a comprehensive list of all stocks along with their full financial data.
    - Use get_investortdata function to get the portfolio of these investors
    - Function to get the list of superstars of investing get_data.
